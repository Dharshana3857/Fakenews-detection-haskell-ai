# Fakenews-detection-haskell-ai
Hybrid Fake News Detection System using Functional Programming and Machine Learning

## Description
Hybrid Fake News Detection using:
- Haskell Functional Programming
- Machine Learning (Python)

## Features
- Keyword & Clickbait Analysis (Haskell)
- Capital Letter & Punctuation Analysis
- AI Prediction using Naive Bayes
- Confidence Score Output

## Technologies Used
- Haskell
- Python
- scikit-learn
- TF-IDF

## How to Run

### Haskell
ghci
:load fakeNews.hs
main

### Python
python predict_news.py
